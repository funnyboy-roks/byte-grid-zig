# Byte Grid

This is tool built in Zig to take pairs of bytes and convert them into
an image.

This is done by plotting the first byte on the x-axis and the second
byte on the y-axis (origin is top-left).

This image is produced over a compiled "Hello, World!" from Rust:

![./img/out.png](Produced image)

## Development

To build locally, Zig and gcc must be installed and you can build with

```sh
$ make
```
